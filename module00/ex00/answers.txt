# Commande pour afficher la liste des paquets installés, ainsi que leurs versions :
-> pip list

# Commande pour afficher les métadonnées d'un paquet :
-> pip show numpy

# Supprimer un package :
-> pip uninstall numpy

# Installer ou Ré-installer un package : 
-> pip install numpy

# Freeze les packages installés dans un fichier requirements.txt
# (utile pour installation de paquets en dependance) :
-> pip freeze > requirements.txt
